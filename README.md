# OOPs-C-2

C++ repository focused on Object-Oriented Programming examples.

This repo contains C++ programs demonstrating OOP concepts such as classes, objects, inheritance, polymorphism, and encapsulation.

## Contents
- Class and object examples
- Inheritance and polymorphism demos
- Practice problems and short exercises

## How to run
1. Clone the repo:
   git clone https://github.com/shubham-nov06/OOPs-C-2.git
2. Compile a program:
   g++ filename.cpp -o output
3. Run:
   ./output

## Requirements
- A C++ compiler (g++, clang, etc.)
